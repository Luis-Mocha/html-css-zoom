# Esercizio "html-css-zoom"
Riproduco l'interfaccia di una videochiamata "Zoom".
---
Utilizzo in particolare le proprietà **Flexbox** e **Position**: *relative, absolute, fixed*.

Implemento l'utilizzo delle unità di misura:
- **vh** *(viewport height)*
- **vw** *(viewport width)*
